# Pilot Quick Start (Teachers)

## Goal
Run one focused intervention and generate one meeting-ready summary in under 10 minutes.

## 1) Open the teacher workflow
- Go to `teacher-dashboard.html`.
- Select one student from the left caseload list.

## 2) Review the snapshot (30-60 seconds)
- Read **Today’s Focus**.
- Confirm the tier/trend line and recommendation text.

## 3) Launch one action
- Click **Start Recommended Session**.
- Complete one short instructional run.

## 4) Log and return
- Return to the dashboard.
- Confirm the student remains selected.

## 5) Generate meeting output
- Click **Meeting & Reports**.
- Use Summary/Deck/Notes tabs.
- Set language if needed.

## What feedback helps most
- What saved time
- What was unclear
- Where you hesitated
- What felt unnecessary
