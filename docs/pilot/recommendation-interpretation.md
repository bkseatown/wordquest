# Interpreting Recommendations (Pilot Guide)

## What the recommendation means
- It is the system’s best next instructional action based on current evidence and tier logic.
- It is guidance for teacher decision support, not an automatic mandate.

## How to read the focus surface
- **Student name + Tier line**: who needs what intensity now.
- **Reason line**: why this move is suggested now.
- **Primary CTA**: fastest path to action.

## Meeting workspace usage
- **Summary**: teacher-facing concise narrative.
- **Deck**: presentation-ready structure.
- **Notes**: working meeting notes.
- **Export**: handoff outputs.

## Curriculum-aligned interpretation
- Treat alignment labels as mapped support signals for current instructional planning.
- If a mapping is partial, state that clearly in meetings.

## Parent communication guidance
- Keep plain language.
- Emphasize growth and next steps.
- Avoid unexplained acronyms or internal shorthand.
